# Skip: US-004 AC-07 - Delta values formatted consistently with main values

## Acceptance Criterion
Delta values formatted consistently with main values

## Why Not Testable via Shell
This acceptance criterion requires visual UI verification:
- Requires formatting comparison
- Requires currency and decimal place settings
- Requires UI inspection

## Verification Method
This criterion should be verified via:
1. Visual inspection in results panel
2. Verify delta uses same currency/decimal formatting
