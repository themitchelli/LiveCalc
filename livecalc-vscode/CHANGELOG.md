# Changelog

All notable changes to the LiveCalc extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-23

### Added

- Initial extension scaffold
- MGA syntax highlighting for .mga files
- Project configuration via livecalc.config.json
- JSON schema for config file IntelliSense
- Run command (Cmd+Shift+R / Ctrl+Shift+R)
- Initialize Project command
- Status bar integration
- Output channel logging
- Language configuration for MGA (brackets, comments, folding)

### Planned for Next Release

- WASM engine integration
- Data loading pipeline
- Results panel visualization
- Auto-run on save
